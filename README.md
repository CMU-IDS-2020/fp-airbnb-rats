## Setting up the software

### Frontend: 

Required:
- Node

Installing packages: 

```
npm i
``` 

Running:
```
npm run start
```

Building:
```
npm run build
```

### Backend
Installing packages:
```
pip install -r requirements.txt
```
Running:
```
python3 backend2.py
```
